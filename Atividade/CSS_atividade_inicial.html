O que é Tebleless?

Tableless é uma forma de desenvolvimento de sites que não utiliza tabelas para disposição de conteúdo na página, pois defende que os códigos HTML deveriam ser usadas para o propósito que foram criados, sendo que tabelas foram criadas para exibir dados tabulares. A W3C também não desaprova o uso de tabelas, desde que sejam utilizadas para atribuir dados e não para formatar layout. Muitos navegadores travam ou exibem incorretamente formatações usando tabelas.

Dentre as possíveis da metodologia estão a diminuição do peso da página (na realidade, o conteúdo da página é mostrado muito mais rápido, o que não significa que o peso da página seja menor - isso, é claro, sempre dependendo de como o HTML é feito) e melhora da acessibilidade - podem ser visto em dispositivos móveis sem necessidade de um versão especial e por deficientes visuais, embora não sejam muitos os sites tableless que carregam atualmente
